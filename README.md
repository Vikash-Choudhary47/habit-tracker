# 🌿 HabitPro - Daily Routine Tracker

A modern, high-performance habit tracker built with **Glassmorphism UI** and **Vanilla JavaScript**. This app helps you visualize your consistency throughout the month with a clean, responsive dashboard.

## ✨ Features
- **Glassmorphism Design:** Beautiful blur effects and dark mode aesthetics.
- **Dynamic Calendar:** Automatically generates the correct number of days for any month.
- **Daily Progress Bar:** Real-time percentage tracking of your goals for the current day.
- **Local Storage:** Your habits and progress are saved locally in your browser—no login required.
- **Responsive Layout:** Works perfectly on both Desktop and Mobile devices.

## 🛠️ Built With
- **HTML5:** Semantic structure for the dashboard.
- **CSS3:** Custom animations (Ken Burns effect), Flexbox/Grid, and Backdrop-filters.
- **JavaScript (ES6):** Logic for month navigation, progress calculation, and data persistence.

## 🚀 How to Use
1. **Add a Goal:** Type your habit in the input bar and click "Add Goal".
2. **Track Progress:** Click the checkboxes for each day you complete a task.
3. **Navigate:** Use the arrows to look back at previous months or plan for future ones.
4. **Delete:** Click the ❌ icon to remove a habit and its data permanently.
